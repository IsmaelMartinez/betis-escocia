## Typography

| Family | CSS Variable | Usage |
|--------|--------------|-------|
| **Display** | `--font-display` | Heroes, impact headlines |
| **Heading** | `--font-heading` | Section headers |
| **Body** | `--font-body` | Body text, paragraphs |
| **Mono** | `--font-mono` | Scores, data, code |

## Pattern Library

| Pattern | Class | Usage |
|---------|-------|-------|
| Verdiblanco | `.pattern-verdiblanco` | Full stripe pattern |
| Verdiblanco Subtle | `.pattern-verdiblanco-subtle` | Light texture |
| Tartan Subtle | `.pattern-tartan-subtle` | Scottish texture |
| Edinburgh Mist | `.bg-edinburgh-mist` | Atmospheric gradient |

## Spacing

Based on Tailwind CSS 4px spacing scale: `1=4px`, `2=8px`, `4=16px`, `6=24px`, `8=32px`

## Border Radius

| Token | Value | Usage |
|-------|-------|-------|
| `rounded-lg` | 8px | Buttons, inputs |
| `rounded-xl` | 12px | Small cards |
| `rounded-2xl` | 16px | Cards, containers |
| `rounded-3xl` | 24px | Hero elements |
